# fyk’s blog

最近一个学期一直很迷茫，思考着我人生的方向与意义，但终究是没有结果的。过往的价值体系全然崩塌，高中那套唯分数论的幼稚想法早已过时。那我究竟应该干些什么呢？

这个问题我问了好些人，大抵得到的答案是去找到自己热爱的事，并为之而奋斗。那么我究竟“热爱”什么呢？我或许是喜欢数学，物理，计算机之类的，又或许是不讨厌罢了。总之，我对任何一件事的喜爱程度都到不了“热爱”的程度。如此，我总是想着找到我真正热爱的事物再出发，想好了再做。

但想全是问题，做才是答案。意义不会在空想中产生，它需要在做之中产生。我过去兴许过分追求结果的价值，而忽视了努力达成目标过程中超越自己的价值。过去中，我所接受的过程中的价值的概念很多时候是能力上的提升，例如“参加比赛输了也不要紧，至少培养了能力嘛”。但仔细想培养的能力有什么用，这仍是一种功利主义的导向。它归根结底还是追求着所谓之“有用”，然而把短期评价体系替换为长期评价体系并不能解决为什么有价值这个根本性问题。

空谈了这么多，我归根结底还是想去寻找自己的价值，而做热爱的事又和价值有何关系呢？我能想到的解释是，做热爱的事给人快乐与动力，进而更容易实现虚无缥缈的价值。因而，现在的关键点就是价值的找寻。高中时特别喜欢西西弗斯的故事，永远掉落的巨石和“应当设想西西弗斯是快乐的”构成了巨大的割裂。这种快乐源于超越自己的价值。这种价值不是所谓“有用的能力”提升，而是在过程中对自身生命的发掘，对自身存在的确认。或许荒谬世间的一切目的都毫无意义，那人类为何还要“激情的舞蹈”。

“人是一团无用的激情。”

“为什么要攀登那座山？”
“因为它就在那里。”

做一件事不需要任何其他理由，“我想做”便足够。意义与价值来源于过程中对生命存在的确认，来源于你面对种种挑战的反应。艾利克斯·霍诺德徒手登顶台北101大厦的价值来源于战胜恐惧的每个瞬间，黑天朝日马拉松比赛的意义藏在想要放弃得到每个瞬间。不要有太多顾虑，不要担心结果的价值。真正热爱的事或许没有，但“我想要”的事可不少，去开启我的地球online之旅吧。

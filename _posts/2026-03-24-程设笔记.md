# 程序设计实习笔记
## 类与对象1
### 引用
 普通引用 int & a = b; 在初始化后引用和原变量就捆绑在一起，多个也一样

常引用 const int & a = b; 在引用的基础上，可以通过修改b，修改a，b 但不能通过a修改a,b 使得程序的安全性更好

常引用能被引用和变量赋值，但不能给引用赋值（const类型变量也不行），除非类型强制转换
e.g. int n = 200;  const int & r = n; int & t = (int&)r;

wrong example: int & r = n * 5; 表达式传递回来的是临时对象，临时对象不能产生引用

### const 关键字的应用
常值 const int n = 100; const string s = "1145141919810";

常量指针 const int * p = &n; 与常引用一样不能通过这个对象（指针）修改原变量的值，不同的是，常指针的指向可以变化
同样，常量指针无法给一般指针赋值（本质上是不让通过这个常指针，产生可能修改原变量的可能性）

### 动态内存分配
new的用法: int *p = new int; int *p = new int[114514] 从0-114514-1（可访问的下标）
注意必须用delete  delete p; delete [] p;

### 函数的重载和缺省
重载，名称相同，参数的个数和类型不同，自动调用
缺省 void func(int x, int y - 1, int z = 2) 从右往左连续缺省若干个

### 类与对象基本概念
class 成员默认和继承都是 private, struct 则是成员默认和继承都为public

使用类中对象的方法：1. 变量名字(or引用名).成员名 2.指针->成员名

成员函数要么在class里面正常写完，

要么在class内部定义void init();

在外面写 void 类名::init() {具体定义}

注意：当你将声明和实现分开时，必须把缺省值写在类内部的声明中，而类外的实现部分不能再写缺省值。
